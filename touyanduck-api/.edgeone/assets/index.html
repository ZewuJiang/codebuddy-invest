⚡ 动态 API 端点

GET /api/briefing
获取最新简报的结构化摘要（精简版）
GET /api/dates
获取所有可用的历史数据日期列表
GET /api/archive?date=2026-04-08
获取指定日期的完整简报数据
参数: date (YYYY-MM-DD) | type=summary|full (默认 summary)
GET /api/trend?days=30
获取最近 N 天的趋势数据（情绪分/风险分/VIX/标普）
参数: days (7/14/30，默认 7)
GET /api/smart-money?fund=伯克希尔
查询指定基金的历史操作记录
参数: fund (基金名) | 不传则返回所有基金最新动态
GET /api/search?q=NVDA
在历史简报中搜索关键词
参数: q (搜索关键词) | limit (返回条数，默认 10)
GET /api/predictions
获取最新的预测市场数据（Polymarket / CME FedWatch）

🔧 管理端点

POST /api/sync
数据同步端点 — 接收每日产出数据并写入 KV 存储
需要 Authorization header 鉴权

💡 使用示例

# 获取最新简报
curl -s https://YOUR_DOMAIN/briefing.md

# 获取历史数据
curl -s "https://YOUR_DOMAIN/api/archive?date=2026-04-08"

# 查看趋势
curl -s "https://YOUR_DOMAIN/api/trend?days=30"

# 搜索关键词
curl -s "https://YOUR_DOMAIN/api/search?q=VIX"
