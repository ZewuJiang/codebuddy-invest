# 🦆 投研鸭 Briefing Skill — OpenClaw 安装与使用指南

## 一、这是什么？

**投研鸭 Briefing** 是一个 OpenClaw / ClawHub Skill，安装后可以让 AI 助手自动获取并解读每日二级市场策略简报。

简单说：**安装一次，以后每天问一句「今天市场怎么样」，AI 就能给你一份专业级投资简报。**

覆盖内容：
- 🇺🇸 美股三大指数 + M7 七巨头
- 🌏 亚太市场（A股/港股/日经/韩国）
- 🛢️ 大宗商品（黄金/原油/美元/美债/人民币）
- ₿ 加密货币（BTC/ETH）
- 📊 GICS 11 板块热力图
- 💰 聪明钱动向（桥水/伯克希尔/ARK/段永平）
- 🔮 预测市场（Polymarket/CME FedWatch）
- 🚦 7 项安全信号红绿灯 + 风险评分
- ⚡ 行动建议 + 本周前瞻事件

---

## 二、如何安装

### 方法 1：ClawHub 在线安装（推荐）

> 如果 Skill 已发布到 ClawHub，直接搜索 `touyanduck-briefing` 点击安装即可。

### 方法 2：手动安装

1. 打开 OpenClaw 设置
2. 找到 Skills 管理
3. 点击「添加 Skill」
4. 将 `SKILL.md` 文件内容粘贴进去
5. 保存

---

## 三、如何使用

安装完成后，直接跟 AI 说以下任意一句话：

| 你说的话 | AI 会做什么 |
|---------|-----------|
| 「今天市场怎么样」 | 给你完整的每日策略简报 |
| 「美股情况」 | 提取美股指数 + M7 数据 |
| 「聪明钱在干嘛」 | 展示桥水/伯克希尔/ARK 等动向 |
| 「NVDA 怎么样」 | 提取英伟达的详细分析 |
| 「风险大吗」 | 展示 7 项红绿灯安全信号 |
| 「这周关注什么」 | 展示本周前瞻事件日历 |
| 「market briefing」 | 英文同样可以触发 |

**不需要任何额外设置，问就行。**

---

## 四、数据更新频率

- **每工作日全量采集** 1 次（覆盖所有维度）
- **每 4 小时刷新行情**（价格/涨跌实时更新）
- 数据来源：Google Finance / AkShare / 英为财情 / OilPrice / CoinGecko / Polymarket / SEC EDGAR

## 六、技术架构（给懂技术的人看）

```
每4小时自动运行 → 产出4个JSON → 渲染为 Markdown → 部署到 GitHub Pages
                                                    ↓
                                        https://zewujiang.github.io/touyanduck-api/
                                                    ↓
                                        OpenClaw Skill (curl URL → 读MD → 回答)
```

**公网端点**（GitHub Pages，全球 HTTPS）：
- `/briefing.md` — 完整 Markdown 简报（AI 主要读这个）
- `/briefing.json` — 核心简报 JSON
- `/markets.json` — 市场详情 JSON
- `/watchlist.json` — 标的分析 JSON
- `/radar.json` — 风险雷达 JSON
- `/meta.json` — 元数据（版本、更新时间）

## 七、常见问题

**Q: AI 说数据过期了怎么办？**
A: 简报每工作日更新。如果在周末或假日，数据会显示为上一个工作日的。

**Q: 我问了一个股票但 AI 说不在监控列表？**
A: 目前监控约 15 只核心标的（M7 + AI链 + 聪明钱重仓）。如需增加，联系管理员。

**Q: 数据准确吗？**
A: 数据经过「八大铁律 + 29 条致命错误零容忍」质量体系保障，价格来源为 Google Finance / AkShare 实时行情。但仍建议以交易所实时数据为准。
