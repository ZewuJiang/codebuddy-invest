## 一、整体原理

现在的流程是这样的：

```
你说"投资Agent" 
  → CodeBuddy 生成日报 (MD + PDF) 
  → [新增] 提取数据为JSON 
  → [新增] 上传到微信云数据库
  → 大老板打开小程序 → 看到真实数据 🦆
```

**你只需要做一次配置（大约30分钟），之后每天自动同步。**

## 二、Step 1：开通微信云开发

### 什么是"云开发"？

简单说，就是微信官方提供的一个"免费数据库"。我们把日报数据存到这个数据库里，小程序就能读到真实数据了。

### 操作步骤

1. **打开微信开发者工具**（就是你之前预览小程序用的那个工具）

2. **打开投研鸭项目**（`touyanduck_appid` 文件夹）

3. **点击工具栏上的「云开发」按钮**
   - 位置：顶部工具栏，一般在"编译"和"真机调试"旁边
   - 图标是一朵小云 ☁️

4. **在弹出的窗口中点击「开通」**
   - 选择「按量计费」（有免费额度，你的用量远远用不完）
   - 环境名称填：`touyanduck`（或者随便起个名字）
   - 点击确认

5. **等待环境创建完成**（约1-2分钟）

6. **记下你的「环境ID」**
   - 创建完成后，在云开发控制台右上角会显示环境ID
   - 格式类似：`touyanduck-xxxxx` 或 `cloud1-xxxxxxx`
   - **把这个ID记下来**，后面要用！

> 💡 **免费额度有多少？** 
> 基础版每月100万次数据库读写。你的日报每天只写4条、大老板看几次，
> 连免费额度的0.1%都用不到，完全不用担心费用。

## 三、Step 2：创建数据库集合

### 什么是"集合"？

你可以把它想象成Excel里的4张表。每张表存一类数据。

### 操作步骤

1. 在云开发控制台，点击左侧「数据库」

2. 点击「添加集合」按钮，依次创建以下4个集合：

   | 集合名称 | 对应小程序页面 | 存什么数据 |
   |---------|-------------|-----------|
   | `briefing` | 简报页 | 核心事件、判断、行动建议、情绪分数 |
   | `markets` | 市场页 | 美股、M7、亚太、大宗、加密、GICS板块 |
   | `watchlist` | 标的页 | 各板块个股数据 |
   | `radar` | 雷达页 | 红绿灯、风险评分、聪明钱动向 |

3. **设置权限**：点击每个集合右侧的「权限设置」
   - 选择「所有用户可读，仅管理端可写」
   - 这样大老板能读数据，但不能改数据（只有你的脚本能写）

> ⚠️ 集合名称必须完全匹配（全小写，没有空格），否则数据读不到！

## 四、Step 3：获取密钥并配置

### 获取 AppSecret

1. 打开浏览器，访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 用你的微信扫码登录（管理员账号）
3. 左侧菜单找到：**开发管理** → **开发设置**
4. 找到「AppSecret(小程序密钥)」，点击「生成」
5. **用微信扫码确认**，然后复制 AppSecret
6. **妥善保存这个密钥！** 关闭窗口后就看不到了

> ⚠️ AppSecret 非常重要，相当于你的银行卡密码。千万不要分享给别人！

### 配置环境变量

打开终端（Mac上按 `Cmd + Space`，输入"终端"，回车），执行以下命令：

```bash
# 把下面的 xxx 替换成你的真实值
echo 'export WX_APPID="wx9e7317dd1e9b88aa"' >> ~/.zshrc
echo 'export WX_APPSECRET="8693890d1301ab28d013902083e569f5"' >> ~/.zshrc
echo 'export WX_CLOUD_ENV="cloud1-3g6wj06h84f38ea8"' >> ~/.zshrc

# 让配置生效
source ~/.zshrc
```

### 验证配置

```bash
# 检查是否设置成功
echo $WX_APPID
echo $WX_APPSECRET
echo $WX_CLOUD_ENV
```

如果三行都有输出值，说明配置成功 ✅

## 五、Step 4：修改小程序配置

打开 `touyanduck/app.js`，找到这两行：

```javascript
useCloud: false,           // 改成 true
cloudEnvId: '',            // 填入你的云环境ID
```

改成：

```javascript
useCloud: true,
cloudEnvId: '你的云环境ID',  // 例如 'touyanduck-xxxxx'
```

**就这一个地方要改，其他代码都不用动！**

## 八、常见问题

### Q: 上传失败怎么办？

不影响日报产出。上传是"旁路"操作，失败了小程序会继续用Mock数据或上一次的缓存。

### Q: 大老板打开小程序看到的是旧数据？

可能是你今天还没跑日报。小程序会优先显示当天数据，如果没有就显示最近一天的。

### Q: 要花钱吗？

微信云开发基础版完全免费，每月100万次读写。你的场景连1%都用不到。

### Q: 我想改回Mock数据怎么办？

把 `app.js` 里的 `useCloud: true` 改回 `useCloud: false` 就行了。

### Q: 数据安全吗？

数据在微信生态内流转（你的电脑 → 微信服务器 → 小程序），不会暴露到公网。AppSecret 存在你本地环境变量里，不会上传到代码仓库。

### Q: 云环境ID在哪里看？

微信开发者工具 → 点击「云开发」按钮 → 右上角显示的就是环境ID。
