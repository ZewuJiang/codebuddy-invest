# 2026-04-09 工作日志

## 09:07 - 修复 app-sync 自动触发超时问题

- **需求/背景**：launchd 定时任务每次都在正常触发，但 codebuddy CLI 执行耗时 15-30 分钟不等，之前 TIMEOUT=1800（30分钟）仍不够用，导致 4/9 06:00 任务被超时强制终止
- **产出**：
  - `scripts/app-sync.sh` — TIMEOUT 从 1800 改为 3600（1小时），修正注释
  - `~/Library/LaunchAgents/com.codebuddy.appsync.plist` — 新增 RunAtLoad + KeepAlive 配置
- **关键决策/技术要点**：
  - 根因是超时值不足，不是 launchd 未触发
  - 添加 RunAtLoad=false（不在登录时立刻执行，但保证任务注册）
  - 添加 KeepAlive.SuccessfulExit=false（非正常退出时 launchd 自动重试）
- **验证**：手动触发一次，845秒完成，validate 50/50 PASS，全流程成功
