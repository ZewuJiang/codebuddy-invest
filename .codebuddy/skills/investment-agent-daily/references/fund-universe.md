## 七、batch 4扫描指南

> **目的**：系统化扫描头部基金动态，确保不遗漏重要信息

### 扫描流程（3-5次web_search）

1. **13F/持仓变动扫描**（1次）→ 搜索"hedge fund 13F filings latest" / "大型基金持仓变动"，重点关注一级核心8家
2. **策略师/大师发言扫描**（1次）→ 搜索"Warren Buffett latest" / "Ray Dalio latest" / "Cathie Wood latest" / "Larry Fink latest" 等，捕捉核心人物最新公开观点
3. **北向/南向资金+外资动向**（1次）→ 搜索"北向资金今日" / "南向资金" / "外资持股变动"
4. **基金业绩/行业动态**（按需0-1次）→ 搜索"hedge fund performance latest" / "基金排名" / "对冲基金回报"
5. **知识库联动检查** → 扫描完毕后，对照本知识库三梯队清单，确认一级核心机构中有重大动态的均已纳入§4

### 「重大动态」判定标准

以下任一条件满足即视为重大动态，必须纳入§4报告：
- **13F季度披露**（每年2/5/8/11月中旬）中一级核心机构的重大调仓（增减持>10%或新建仓/清仓）
