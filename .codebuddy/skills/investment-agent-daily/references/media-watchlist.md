# 全球财经媒体追踪清单（v1.0）

## 四、batch 0 / 0a 扫描 SOP

### batch 0（每日必扫，2-3 次搜索）

```
搜索1: 英文头条综合扫描
  → "Bloomberg Reuters WSJ markets today {date}" 
  → 确认全球重大新闻头条

搜索2: 中文头条综合扫描
  → "华尔街见闻 第一财经 智通财经 格隆汇 今日要闻"
  → 确认中国/港股/A股重大新闻

搜索3（可选）: 交叉验证
  → 比对中英文扫描结果，确保无信息遗漏
  → 特别关注：中文媒体独有的中国政策新闻 + 英文媒体独有的美股/全球新闻
```

### batch 0a（深度补充，1-2 次搜索）

```
搜索1: 美股+AI/科技深度
  → "CNBC MarketWatch today" + "AI startup funding news today"
  → 覆盖美股盘中盘后重大报道 + AI/科技产业动态

搜索2（可选）: 中国深度
  → "36Kr AI 融资" + "晚点 LatePost 今日"
  → 覆盖中国科技创投 + 互联网深度
```

```
IF 油价日内波动 > 3% → 扫描 OilPrice.com / 金投网
IF BTC 日内波动 > 5% → 扫描 CoinDesk / The Block
IF 日元/日股异常波动 → 扫描 Nikkei Asia

IF 重大监管政策发布 → 扫描 SEC.gov / 中国证监会
```

## 五、交叉校验机制

> **核心不在于追踪所有媒体，而在于交叉验证。**

### 1. 中英文双语交叉

- **英文媒体**（Bloomberg/Reuters/WSJ）：捕捉美股/全球/地缘新闻
- **中文媒体**（华尔街见闻/智通财经/第一财经/格隆汇）：捕捉中国/港股/政策新闻
- **互补覆盖**：英文媒体不一定覆盖中国政策细节，中文媒体不一定覆盖欧美小公司突发事件

### 2. 通讯社 + 深度媒体交叉

- **速度层**：Reuters/Bloomberg（快但浅）
- **深度层**：FT/Barron's/The Information/晚点 LatePost（慢但深）
- **互补**：速度层确保不遗漏，深度层提供投资含义分析

### 3. 终审自查步骤

报告撰写完成后，快速扫描当日头条：
1. 检查 Bloomberg/华尔街见闻 Top 5 头条是否都在报告中有所体现
2. 检查是否有"只出现在某一语言媒体"的重大新闻被遗漏
3. 检查 AI/科技领域是否有重大融资/产品发布未被 batch 0b 捕捉
