## 核心能力

### 1. 数据准确性保障

- **三级数据来源分级**:
  - ⭐⭐⭐ SEC官方文件(Form 10-K, 10-Q, 8-K) - 最高权威
  - ⭐⭐⭐ 交易所官方公告(联交所、上交所、深交所)
  - ⭐⭐⭐ 公司官方投资者关系网站
  - ⭐⭐ 权威第三方(Bloomberg, FactSet, Bullfincher)
  - ❌ 非官方网站(如CompaniesMarketCap) - **严格禁止**

- **强制三方交叉验证**:
  - 所有关键财务数据必须从至少3个来源验证
  - 差异>1%必须重新验证
  - 最终数据必须注明所有来源

- **财年(FY) vs TTM数据辨析**:
  - 历史对比必须使用财年数据
  - TTM数据仅用于实时追踪
  - 博通案例教训:CompaniesMarketCap使用TTM导致6-15%误差

### 2. 分析框架(4阶段)

**阶段1: 数据收集与验证(90分钟)**
- 官方数据源定位(SEC/交易所)
- 最新季度核心财务数据提取
- 历年营收数据收集(10年)
- 三方交叉验证

**阶段2: 业务结构分析(60分钟)**
- 分部营收拆解
- 地区/产品营收拆解
- 关键驱动因素识别

**阶段3: 管理层信息提取(45分钟)**
- 下季度指引(Guidance)解读
- 全年展望提取
- 核心亮点总结(Top 5,数据支撑)

**阶段4: 风险与投资建议(45分钟)**
- 风险矩阵构建(量化风险)
- 投资价值评估
- 目标价计算(含敏感性分析)

## 反面案例与教训

### 案例1: 博通历年营收数据错误(2026-04-29)

**问题**:
- 原数据来源:CompaniesMarketCap
- 错误类型:使用TTM数据,而非财年数据
- 误差:FY2025营收误差-6.4%($4.39B差异)

**纠正过程**:
1. ✅ 用户质疑"历年营收请参考官方的财年报告披露"
2. ✅ 从SEC Form 10-K重新验证10年数据
3. ✅ 从博通官方IR确认(CEO确认FY2024营收$51.6B)
4. ✅ 从Bullfincher、Stock Analysis交叉验证
5. ✅ 创建独立验证报告记录全过程

**教训**:
- ✅ SEC官方文件 > 公司官方公告 > 权威第三方 > 财经媒体
- ✅ 财报分析必须使用财年数据,而非TTM数据
- ✅ 三方交叉验证必不可少
- ✅ 用户直觉的价值(专业投资者的质疑往往准确)

**影响**:
- CAGR从16.8%(错误)修正为18.63%(正确)
- 累计增长率从+337.7%(错误)修正为+382.6%(正确)
- CompaniesMarketCap已列入**禁止使用**数据源
