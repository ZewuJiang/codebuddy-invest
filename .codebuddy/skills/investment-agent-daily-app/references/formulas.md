## 一、红绿灯 trafficLights 7项计算规则

| 指标 | 采集来源 | 绿（green） | 黄（yellow） | 红（red） | threshold 文字 |
|------|---------|------------|-------------|----------|---------------|
| VIX波动率 | Google Finance `VIX:INDEXCBOE` | < 18 | 18 ~ 25 | > 25 | `<18绿 / 18-25黄 / >25红` |
| 10Y美债收益率 | FRED `DGS10` / web_search | < 4.0% | 4.0% ~ 4.5% | > 4.5% | `<4.0%绿 / 4.0-4.5%黄 / >4.5%红` |
| 布伦特原油 | OilPrice.com / web_search | < $90 | $90 ~ $110 | > $110 | `<$90绿 / $90-110黄 / >$110红` |
| 美元指数DXY | Investing.com / web_search | < 102 | 102 ~ 107 | > 107 | `<102绿 / 102-107黄 / >107红` |
| HY信用利差 | FRED `BAMLH0A0HYM2` / web_search | < 4% | 4% ~ 5% | > 5% | `<4%绿 / 4-5%黄 / >5%红` |
| 黄金XAU | web_search / OilPrice.com | < $2,200 | $2,200 ~ $3,500 | > $3,500 | `<$2200绿 / $2200-3500黄 / >$3500红` |
| 离岸人民币CNH | web_search / AkShare | < 7.15 | 7.15 ~ 7.30 | > 7.30 | `<7.15绿 / 7.15-7.30黄 / >7.30红` |

> ⚠️ AI 填写 trafficLights 时：`value` 写采集到的实际数值（字符串），`status` 严格按上表阈值判断，`threshold` 复制上表「threshold 文字」列。

### 动态迭代机制

**固定3项（永久保留）**：VIX波动率、10Y美债收益率、黄金XAU

**动态4项（随市场环境调整）**：布伦特原油、美元指数DXY、HY信用利差、离岸人民币CNH

**备选指标池**：铜价 / 2Y-10Y利差 / 美联储降息概率 / BofA牛熊指标 / 天然气

**迭代规则**：每次全量执行时评估7项是否仍为最优组合。若某项已长期绿灯且与当前核心风险无关，可在下次执行时提议替换（需在交付确认中说明替换理由）。

## 二、riskScore / riskLevel / riskAdvice 计算公式

### riskScore 公式

```
基础分 = 30（代表市场正常背景噪音）
权重：green = 0分 / yellow = 10分 / red = 20分
riskScore = min(100, 30 + Σ各灯颜色权重)

示例：7绿 → 30+0=30分；3黄4绿 → 30+30=60分；7红 → 30+140→封顶100分
```

### riskLevel 规则

```
riskScore < 45  → "low"
riskScore 45-64 → "medium"
riskScore ≥ 65  → "high"
```

### riskAdvice 内容规范（v5.4 — 去操作化）

```
格式：动态生成，禁止模板化套话。
要求：
  ①点名最危险的 1-2 项红灯/黄灯指标（说清楚危在哪里）
  ②综合底层指标（VIX/Put-Call/信用利差等）判断情绪方向，禁止机械引用 F&G 单一指标值
  ③指向本周最关键的催化剂/时间节点
  ④禁止包含操作建议（仓位/加仓/减仓/对冲等）
  ⑤不超过 2 句话，每句有信息量

正确示例：
  "黄金$4,627亮红灯（避险情绪极端），布伦特$109逼近红灯区间——若周四CPI确认通胀二次抬头，当前反弹基础将被抽空。"

错误示例（禁止）：
  "布伦特$109叠加F&G仅19（极度恐惧），建议维持5成仓位+能源对冲，等CPI落地后再评估。"
```

## 三、metrics 综合评级计算规则

> 适用于 watchlist 每只已上市标的的 metrics[5]（综合评级）

**`calc_star_rating(change, pct_30d)`**：

| 评级 | 触发条件 |
|------|---------|
| ⭐⭐⭐⭐⭐ | 30日涨超+15% 且 单日为正 |
| ⭐⭐⭐⭐ | 30日涨+5%~+15%，或单日涨超+3% |
| ⭐⭐⭐ | 30日在-5%~+5%（震荡整理） |
| ⭐⭐ | 30日跌-5%~-15% |
| ⭐ | 30日跌超-15%（深度回调） |

## 四、sentimentScore → sentimentLabel 映射

| score 区间 | sentimentLabel |
|-----------|---------------|
| 0 - 20 | 极度恐惧 |
| 21 - 40 | 偏恐惧 |
| 41 - 60 | 中性 |
| 61 - 75 | 偏贪婪 |
| 76 - 90 | 贪婪 |
| 91 - 100 | 极度贪婪 |

> ⚠️ 禁止使用非枚举值（如「偏悲观」「偏乐观」）。sentimentScore 必须基于 VIX/Put-Call/信用利差等底层指标独立综合打分，禁止简单对齐 CNN F&G。
